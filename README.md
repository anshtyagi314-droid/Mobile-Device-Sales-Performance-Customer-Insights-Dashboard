# Mobile-Device-Sales-Performance-Customer-Insights-Dashboard
🧾 Project Overview

This project is an interactive Sales Analytics Dashboard built using Microsoft Power BI. It focuses on analyzing Samsung mobile sales data across different dimensions such as time, location, brand, payment methods, and customer behavior.

The dashboard provides clear, visually engaging insights that help in understanding sales performance, customer preferences, and market trends.

🚀 Features
🔹 Key Metrics (KPIs)
Total Sales: 42K
Total Quantity Sold: 88
Total Transactions: 40
Average Unit Price: 596.18
📅 Time-Based Analysis
Monthly sales comparison (January vs February)
Day-wise sales trends to identify peak sales days
🌍 Geographic Insights
Interactive map showing Total Sales by City
Highlights top-performing cities like:
Seattle
Denver
Milwaukee
San Jose
Virginia Beach
📱 Product Analysis
Sales distribution by Brand:
Apple
Samsung
Google
OnePlus
Xiaomi
Quantity sold by Mobile Models
💳 Payment Insights
Transactions segmented by payment methods
Helps identify customer payment preferences
⭐ Customer Feedback
Customer ratings visualization
Displays satisfaction levels across rating scales
🎯 Objectives
Analyze overall sales performance
Identify top-selling brands and models
Understand customer purchasing behavior
Track performance across cities and time
Provide actionable insights for business decisions
